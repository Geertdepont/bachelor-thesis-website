Employ well-defined and accepted coding standards, as they improve both code understandability and maintainability.


Why?

Using well-defined coding standards is necessary when collaborating. When non-logical variable or function names are used, it becomes harder for others to understand the code. Therefore use coding standards. https://github.com/ribot/android-guidelines/blob/master/project_and_code_guidelines.md
